# Weekly Calendar Planner

A Python-based weekly calendar planner application.

## Description

This project aims to create a flexible weekly calendar planner that can handle recurring events and generate visual schedules.

## Planned Features

- CSV-based event data input
- Support for recurring events (daily, weekly, bi-weekly)
- Calendar visualization
- Web API for event submission

## Tech Stack

- Python 3.x
- Flask (planned)
- Data visualization library (TBD)

## Status

Work in progress - initial setup phase.
